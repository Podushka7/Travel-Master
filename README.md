# ИС «Travel-Master» (Автоматизация турагентства)

**Travel-Master** — это автономное Desktop-приложение, разработанное для малых туристических агентств. Программа обеспечивает мгновенную работу с базой путевок и гарантирует локальную безопасность данных без необходимости подключения к интернету или покупки облачных подписок.

## Технологический стек
*   **Язык:** C++17
*   **Фреймворк:** Qt 6 (Widgets, Core, GUI)
*   **Система сборки:** CMake
*   **Хранение данных:** Кастомная бинарная сериализация (`.tdb`)
*   **Паттерн проектирования:** Model-View

## Ключевой функционал
*   Полный цикл управления карточками туров (Добавление, Чтение, Редактирование, Удаление).
*   Строгая валидация вводимых данных на уровне интерфейса (цены, время тура, место отдыха).
*   Мгновенный отклик интерфейса за счет хранения активной базы в оперативной памяти (QMap).
*   Сохранение и загрузка зашифрованных бинарных файлов (с возможностью частичного импорта N-записей).
