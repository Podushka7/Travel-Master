## Цель проекта
Разработка приложения для туристических агентств.


## Границы проекта
Мы реализуем приложение, которое позволит туристическим агентствам:
*   Создавать туры в различные места
*   Устанавливать цену и время отдыха
*   Указывать, требуется ли виза для отдыха
*   Фильтровать туры по определённым характеристикам

## Принципы работы
*   Проект будет выполнен на кроссплатформенном фреймворке **Qt**
*   Программа будет написана на языке **C++**
*   В качестве методологии управления будет использован фреймворк **Scrum**
